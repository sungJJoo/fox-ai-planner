# FOX AI 연구소 담당표

3인 팀(팀장 1 · 연구원 2)의 **3주 순환 근무 담당표 + 업무 관리 + 근무 일정 + 연간 캘린더**를 한 화면에 모은 웹 대시보드입니다.
Google Sheets를 데이터베이스로, Google Apps Script를 백엔드로, GitHub Pages를 호스팅으로 쓰는 **서버리스 구조**라 운영비가 0원입니다.

🔗 **사이트**: https://sungjjoo.github.io/fox-ai-research-schedule/

---

## ✨ 주요 기능

### 담당표 · 멤버
- **3주 순환 담당표** — 앵커 날짜 기준으로 3주 패턴이 무한 반복, 오늘 자동 강조
- **멤버 관리** — 추가/수정/삭제, 8색 색상 슬롯. 이름 변경 시 담당표·근무일정·완료업무·반복업무 시트에 자동 반영

### 업무 관리
- **업무 추가/수정/삭제** — 모달에서 바로, 시트 열 필요 없음
- **담당 다중 선택** — 여러 명 또는 "AI 연구원(전원)" 지정 가능
- **개별 완료 체크** — 담당자가 여러 명이면 각자 체크, 전원 완료 시 자동으로 완료 처리
- **마감 카운트다운** — D-7 ~ D-1, TODAY, 지남까지 색상 단계별 배지
- **누락 업무 알림** — 기한 지난 미완료 업무를 모아 보기
- **댓글** — 업무별 댓글, 작성자 선택 필수(익명 불가)

### 완료 업무
- **30분 후 자동 아카이브** — 완료 체크 30분 뒤 '완료 업무' 시트로 자동 이동
- **담당자 · 기간 필터** — 이번 주/이번 달/최근 7·30일
- **타이밍 배지** — 마감 대비 빠름/당일/지각 표시
- **완료 업무 수동 추가** — 시스템 도입 전 끝낸 업무도 기록 가능

### 반복 업무
- **요일별 담당** — 월~토 각 칸에 사람 이름 또는 활동 키워드(영어, 미술 등) 지정
- 오늘 요일 자동 강조

### 근무 일정
- **월 자동 전환** — 매달 1일 기준으로 그 달(6월→7월→8월)로 날짜 자동 교체. 근무패턴·색상·테두리는 그대로 유지
- 월~토 레이아웃(1일이 일요일이면 2일부터), 6주 필요한 달은 자동 확장
- **연차·반차 계획** — 근무일정에서 자동 추출, 7일 이내 강조

### 연간 캘린더 (2026~2027)
- **행사 · 공휴일 · 계획** 3종 색상 구분, 기간 일정은 연속 막대로 표시
- **한국 공휴일 자동 표시** — 설날·추석·대체공휴일 포함(읽기 전용)
- **연차/반차 자동 표시** — 근무일정에 적은 연차가 캘린더에도 자동으로

### UX · 성능
- **자동 갱신** — 30초마다 변경 감지(해시 폴링), 바뀐 것만 다시 그림
- **즉시 반응(낙관적 UI)** — 추가/수정/삭제가 서버 응답 전에 바로 화면 반영
- **브라우저 알림** — 당일 마감 업무 알림
- **다크 모드** — 시스템 설정 자동 감지 + 토글
- **PWA** — 홈 화면에 앱처럼 설치, 오프라인에서도 마지막 화면 표시
- **단축키** — `N` 업무 추가, `Esc` 닫기

## 🆚 기존 방식과 뭐가 다른가

| | 구글 시트만 사용 | 노션 / 트렐로 등 범용 툴 | **이 대시보드** |
|---|---|---|---|
| **모바일 보기** | 불편(가로 스크롤) | 보통 | 반응형 + PWA로 앱처럼 |
| **3주 순환 · 근무일정** | 수동 계산 | 직접 구성해야 | 팀 워크플로우에 맞춤 내장 |
| **마감 알림 · 카운트다운** | 없음 | 일부 | 색상 배지 + 브라우저 알림 |
| **연차 ↔ 캘린더 연동** | 수동 | 수동 | 자동 |
| **월 날짜 갱신** | 매번 손으로 | 해당 없음 | 매달 1일 자동 |
| **비용** | 무료 | 인원·기능 제한/유료 | 무료 |
| **데이터 소유권** | 내 시트 | 외부 서비스 | **내 구글 시트에 그대로** |

요약: **구글 시트의 데이터 소유권은 유지**하면서, 시트로는 불편한 모바일 UX·알림·필터·자동화를 얹은 형태입니다.

## 🧱 구조

```
[index.html + styles.css + app.js]   ← 프론트엔드 (순수 JS, 프레임워크 없음)
            │  GitHub Pages 호스팅
            ▼  fetch (GET)
[Google Apps Script Web App]         ← 백엔드 API
            ▼
[Google Sheets]                      ← 데이터 (담당표/멤버/근무일정/완료업무/반복업무/캘린더)
```

- `index.html` — HTML 마크업 (PWA 메타 포함)
- `styles.css` — 전체 스타일 (다크 모드 포함)
- `app.js` — 클라이언트 로직 전체
- `manifest.json` · `sw.js` · `icon.svg` — PWA (설치/오프라인)
- `apps-script.gs` — Apps Script 백엔드 코드 백업(실제 실행은 GAS 편집기)

### 성능 최적화
- **버전 카운터 + 해시 폴링** — 30초마다 가벼운 핑으로 변경만 감지 → 폴링 비용 ~80% 절감
- **낙관적 UI** — 추가/수정/삭제 시 로컬 즉시 반영, 실패 시에만 복원
- **GAS 워밍업 + preconnect** — 모달 열 때 미리 연결해 저장 지연 단축

## 🔧 백엔드(GAS) 변경 시

1. `apps-script.gs` 수정
2. [Raw 파일](https://raw.githubusercontent.com/sungJJoo/fox-ai-research-schedule/main/apps-script.gs) 전체 복사 → GAS 편집기에 덮어쓰기 → 저장
3. **배포 → 배포 관리 → 연필 → 버전 "새 버전" → 배포**
4. URL이 바뀌면 `app.js`의 `API_URL` 갱신 후 commit/push

### 트리거 (GAS 편집기에서 1회 실행)
- `installTrigger` — 완료 업무 자동 정리(15분마다)
- `installMonthTrigger` — 근무일정 월 자동 전환(매일 새벽 1시 점검)
